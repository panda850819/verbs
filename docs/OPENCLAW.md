# Panda Verbs on OpenClaw

OpenClaw is unsupported and experimental. Panda Verbs currently ships no
OpenClaw installer, adapter, hooks, or conformance proof.

An external OpenClaw integration may adapt selected skills or delegate work to
a verified Claude Code or Codex installation. OpenClaw must own that routing,
memory, scheduling, and session policy; those concerns do not move into Panda
Verbs.

Do not claim support until the adapter satisfies
[`ADDING_A_HOST.md`](ADDING_A_HOST.md): reproducible install and removal, exact
skill discovery, explicit tool and hook mapping, and one real invocation.
